도메인 모드와 스탠드얼론 모드 중 어느 것을 사용하는가에 따라 프로파일의 의미가 약간 달라짐.

도메인 모드 프로파일은 서버 그룹에 적용되는 설정 옵션 그룹.
프로파일 및 소켓 바인딩 그룹,배포 서버 그룹을 설정함.
관리 콘솔이나 관리 CLI에서 프로파일을 설정할 수 있음.

스탠드얼론 모드 프로파일은 JBoss EAP 구성 전체를 의미.
제공되는 4가지 프로파일이 있어 요구 사항에 가장 적합한 설정을 사용할 수 있음.
필요에 따라 사용자가 새로 작성하여 사용할 수도 있음.
Java EE 6 프로파일은 Java EE 6 API를 의미.

Java EE 6 프로파일은 Java EE 6 API의 하위 집합
Java EE 6는 풀 프로파일과 웹 프로파일의 두 가지 프로파일을 정의함.
또한 추가 프로파일을 만드는 방법도 가능.
JBoss EAP 6는 Full Profile 및 Web Profile을 모두 완벽하게 지원.
