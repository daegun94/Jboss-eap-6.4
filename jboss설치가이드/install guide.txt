1.ubuntu jdk설치(6.4와 호환되는 jdk 8버전 설치!)
sudo apt update
sudo apt install openjdk-8-jdk

https://jboss6.openmaru.io/docs/03.JBossEAP6_Install.html(참고하여 설치)
jboss 설치까지는 좀 다르고
그 이후 관리자등록부터는 같음.
2. jdk java경로 등록하기위한 과정.
which java
ls -al

3. 디렉토리경로 중 java파일 전까지의 디렉토리경로를 저장해둬야함
export JAVA_HOME=/usr/lib/jvm/java-8-openjdk-amd64/jre
export PATH=$PATH:$JAVA_HOME/bin
4. jboss bin파일로 이동(각종 실행파일) $JBOSS_HOME = jboss설치 디렉토리
설치 디렉토리는 마음대로 생성

5.jbossEAP6에서는 보안을 위해서 기본적으로 등록된 관리자가 없는 상태
GUI 방식으로 설치하면 설치 과정에서 관리자 아이디와 패스워드를 설정하게 되ㅣ만,
zip파일을 이용하여 설치할 경우 add-user.sh를 실행하여 초기 관리자 
아이디와 패스워드를 등록해야함.
$JBOSS_HOME/bin 디렉터리로 이동후 ./add-user.sh실행
이후 뒤에 나오는 secret value값을 기록해두어야함(도메인 모드 구성시 필요)

6.스탠드얼론 모드에서 시작과 정지.
bin파일에서 ./standalone.sh
localhost:8080
