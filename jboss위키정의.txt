JAVA EE(Enterprise Edition)는 자바를 이용한 서버측 개발을 위한 플랫폼.
PC에서 동작하는 표준 플랫폼인 JAVA SE에 부가하여
WAS에서 동작하는 장애복구 및 분산 멀티티어를 제공하는 자바 SW기능을 추가한 플랫폼

*****************************JBoss EAP(Enterprise Application Platform 7)특징**************
고가용성 클러스터링,메시징 및 분산 캐싱과 같은 기능에 대해 사전 구성된 옵션을 제공
여기에는 필요할 때만 서비스를 활성화할 수 있는 모듈식 구조가 포함되어 있어
시작속도가 향상
웹 기반 관리 콘솔 인터페이스(CLI)를 사용하면 스크립트를 작성하고 작업을 자동화하는 기능 추가
JBoss EAP에는 Java EE 애플리케이션을 신속하게 개발,배포 및 실행할 수 있는
API 및 개발 프레임워크가 포함되어 있음.
이러한 JAVA EAP용 컨테이너화된 이미지를 제공하여
개발자는 하이브리드 환경에 배포되는 APP를 빠르고 쉽게 구축 및 확장 테스트까지 가능
